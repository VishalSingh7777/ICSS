# ICSS
Integrated Campus Surveillance System (ICSS)

🔗 Live Demo: https://icssv0.netlify.app/

ICSS is a fun frontend project i built for a friends’ team presentation. It showcases the idea of an AI-driven campus surveillance and attendance dashboard using a clean, modern interface.

The project demonstrates how a campus system could manage camera monitoring, student attendance, alerts, analytics, and system settings from one dashboard.

## What It Is

This is a React-based frontend prototype/simulation of an Integrated Campus Surveillance System.
"The frontend design and user interface looks fantastic — I really like it."😁

It includes simulated data for:

- Students
- Cameras
- Attendance logs
- Alerts
- Analytics
- System health
- Camera monitoring views

The goal of this project is to present the concept and user interface of a smart campus monitoring system.

## Important Note

This project does not use real facial recognition, real CCTV feeds, real backend APIs, or a real database.

All data shown in the dashboard is mock/simulated data created for demonstration purposes.

## Features

- Dashboard overview for attendance, cameras, and alerts
- Simulated live camera monitoring interface
- Camera network status page
- Student directory with attendance and risk details
- Attendance records with filters
- Analytics and defaulter tracking
- Alert management interface
- Settings page for model, integrations, notifications, users, and compliance
- Light and dark mode support

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Recharts
- Lucide React
- HTML Canvas for simulated camera visuals

## Project Purpose

This project was created as a frontend concept/demo for an academic or presentation use case. It focuses on UI, workflow, and product structure rather than real AI or production-level implementation.

## Future Scope

If extended further, the system could include:

- Real backend and database
- User authentication
- Real camera stream integration
- Face detection and recognition
- Attendance storage and reporting
